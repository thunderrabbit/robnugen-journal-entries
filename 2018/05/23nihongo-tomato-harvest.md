## 12:07 Wednesday 23 May 2018 JST

コンプリスにタイマーがあります。　There is a timer on Complice.

タイマーが終わったら、トマトをもらいます。　Every time the timer
finishes, I get a tomato.

今日、タイマーの時間を０秒に設定しました。 Today I set the
timer for 0 seconds.

[![2018 may 23 complice harvest](//b.robnugen.com/journal/2018/thumbs/2018_may_23_complice_harvest.png)](//b.robnugen.com/journal/2018/2018_may_23_complice_harvest.png)

０秒で、５千個以上トマトをもらった。 I got over 5000 tomatos.
