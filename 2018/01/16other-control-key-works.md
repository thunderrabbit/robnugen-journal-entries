## 08:44 Tuesday 16 January 2018 JST

このキーボードはちょっと古いからコントロールキーは使えなくなりました。
だから、コントロールキーのひつよ時、他のキーボードのコントロールキーを
つかになりました。

しかし、今日、このキーボードの右のコントロールキーが大丈夫ことがわかり
ました。

嬉しいです。他のキーボードのコントロールキー使えないすむになりました。

This keyboard is old so the control key on this keyboard stopped
working.  Therefore when the control key is important, I have to use
the control key on another keyboard.

But, today, I just realized the control key on the right hand side
works.

I'm happy!  I can get away without using the control key on the other
keyboard.
